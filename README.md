EduBridge - Connecting Minds, Bridging Knowledge
EduBridge is a feature-rich educational platform designed to streamline the connection between students and expert tutors. Whether it's online coaching or offline sessions, EduBridge provides a seamless, secure, and user-friendly environment for both learners and educators to achieve their academic goals.

🚀 Key Features
Smart Tutor Marketplace: Students can discover and filter expert tutors across multiple disciplines such as Mathematics, Physics, and Chemistry.

Integrated Session Management: A secure, private dashboard allows tutors to manage their profiles, update availability, and handle bookings in real-time.

Dynamic CRUD Operations: Built with robust React state management, allowing tutors to update or remove session entries instantly without refreshing the page.

Secure User Authentication: Features a comprehensive login and registration system, ensuring personalized access for students and tutors alike.

Optimized & Responsive UI: Designed with a sleek, modern aesthetic using Tailwind CSS and HeroUI to provide a consistent and professional experience across all screen sizes.

🛠 Tech Stack
Frontend: Next.js (App Router), React.js, Tailwind CSS, HeroUI.

Backend: Node.js, Express.js.

Database: MongoDB.

Environment: Developed and deployed via Vercel.
